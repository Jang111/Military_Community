1.리액트에서의 "DOM"은 가상 DOM을 가리키며, 이것은 웹 페이지의 실제 DOM과 동기화되는 것이 아니라 
리액트의 성능 최적화를 위해 사용되는 가상의 표현입니다. 
이를 통해 리액트 애플리케이션은 빠르게 동작하면서도 유연하게 UI를 업데이트할 수 있습니다.
- 일반적인 의미의 DOM이란
: Document Object Model의 준말로, 문서 객체 모델이다. 이는 넓은 의미로 웹 브라우저가 HTML페이지를 인식하는 방법을 말하며, 좁은 의미로js의 'document'객체와 관련된 객체의 집합을 의미할 수도 있습니다.
2. 웹 개발에서의 '렌더링' : 사용자에게 웹 페이지를 나타내는 것
